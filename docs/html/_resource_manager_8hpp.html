Detailed Description

This file defines the funcionality of managing resources.

The ResourceManager class is a singleton class which provides a base interface which abstracts the general functionality of resource management. Most of the methods have implementations. It allows to load music, font and text. And can also load resources that can be saved in Component class. Once you create a resource in Engine.cpp please add it by using addGameComponent or other add functions. It is efficient to store all the resources into map and use them by calling get functions.
